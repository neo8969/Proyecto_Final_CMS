En este proyecto encontraremos 2 carpetas y 5 archivos en lenguaje de programacion python
La primera carpeta BurgerCMS contiene el ambiente virtual el cual estaremos ocuapando para la generacion del portal
La segunda carpeta con el nombre de Portal_BurgerCMS encontraremos dentro de ella 1 carpeta llamada templates la cual nos servira para
almacenar los html que generamos para el despliegue de nuestra app el cual nos servira para desarrollar el skill de estar creando varias 
ventanas dentro de tu app. Adicional encontraremos nuestro archivo de aplicacion llamado app.py en este encontraremos el codigo en python 
que esta configurado para hacer el despliegue mediante un servidor virtual con el modulo de flask, este esta configurado para poder seleccionar el id 
de un cliente y ya seleccionado poder, seleccionar entre los productos que tenemos a escoger en la base de datos, una vez elegidos los productos
al dar click en el boton finalizar nos generara un ticket en formato txt con el costo total de los productos que elegimos, este txt se estara 
sobrescribiendo cada que hagamos un pedidos nuevo.

Seguido tenemos nuestra base de datos con el nombre Burger_CMS.db la cual creamos para almacenar todos los datos que vayamos recolectando de cleintes
menu y o pedidos.
En el archivo de Burger_CMS.py tenemos el codigo que generamos para la creacion de la base de datos. 
En el archivo clientes.py encontraremos el codigo para poder agregar, modificar o eliminar un cliente segun lo deseemos.
En el archivo main.py intentamos configurarlo para que realizara una cossulta a las funciones de menu, clientes y pedidos para que de esta manera
el usuario pudiera agregar datos de forma automatico desafortunadamente no logre establecer el codigo adecuado para esto.
En el archivo manu vamos a encontrar el codigo para poder ingresar datos a la tabla menu como productos junto con su precio.
Y por ultimo encontraremos el arhcivo de pedido el cual nos servira para ir colecando los pedidos de manera manual y hacer una consulta sobre los pedidos realizados
